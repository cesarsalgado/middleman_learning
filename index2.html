Looking for some challenges?

In this section I list some sites which provide challenges in Computer Science. I haven't tried challenges in all of these sites, but each one seems to be very good.

  • HackerRank: Solve fun programming challenges.
  • Kaggle: Kaggle is an arena where you can match your data science skills against a global cadre of experts in statistics, mathematics, and machine learning.
  • Google Code Jam: Google Code Jam is an international programming competition hosted and administered by Google (source: Wikipedia).
  • CodeChef: CodeChef was created as a platform to help programmers make it big in the world of algorithms, computer programming and programming contests.
  • Project Euler: Project Euler is a series of challenging mathematical/computer programming problems that will require more than just mathematical insights to solve.
  • TopCoder: When you participate in the TopCoder community you gain access to a world of interesting things to do in the form of sophisticated and challenging tournaments, an increasing number of client projects and social interaction with some of the best minds in their respective fields.
Obs: every description for the above sites was copied from their own pages except when mentioned otherwise.
